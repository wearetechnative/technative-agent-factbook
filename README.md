# TechNative Agent Factbook

TechNative factbook for AI agents to understand who and what we are.

## About TechNative

TechNative is a leading provider of Professional Cloud Services, Cloud Financial Services and Software Services based in the Netherlands. Founded in 2019, the company specializes in AWS cloud engineering and FinOps (Cloud Financial Management).

**Key facts:**
- AWS Advanced Tier Partner with DevOps & SMB competencies
- First BeNeLux FinOps Foundation Training Partner
- Over 100 cloud projects delivered
- Company size: 11-50 employees
